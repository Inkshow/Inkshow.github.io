Termux如何使用Hexo搭建自己的博客

发表于 2021-03-11 | 更新于: 2021-03-11 |
Android:三年之期已到，这Windows，不要也罢(￣_,￣ )（bushi

Hexo介绍

Hexo是个快速，简洁且高效的博客框架。Hexo使用Markdown解析文章，在几秒内生成静态网页。

——copy至Hexo官网

搭建环境

  • 首先安装好Termux

  • 切换清华源

    sed -i ‘s@^(deb.*stable main)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main@’ $PREFIX/etc/apt/sources.list
    sed -i ‘s@^(deb.*games stable)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/game-packages-24 games stable@’ $PREFIX/etc/apt/sources.list.d/game.list
    sed -i ‘s@^(deb.*science stable)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/science-packages-24 science stable@’ $PREFIX/etc/apt/sources.list.d/science.list
    apt update && apt upgrade

  • 安装Node.js

    pkg install nodejs

  • 安装git

    pkg install git

  • 安装Hexo

    npm install -g hexo-cli

  • 安装deployer git

    npm install hexo-deployer-git –save

搭建博客

  • 切换目录至包括博客文件夹的路径

    #假如你的博客路径为/storage/emulated/0/Myblog/

    #那么我们要切换的路径就为/storage/emulated/0/

    cd /storage/emulated/0/

  • 初始化博客

    hexo init Myblog

  • 编辑_config.yml

    按照该链接编辑

添加文章

输入 hexo new 文章名 即可

进行编辑时需要Markdown编辑器

更换主题

在伟大的因特网上下载主题后进行如下操作就行

  • 重命名

  • 复制或剪贴进themes文件夹

  • 修改_config.yml

  • 在theme:处修改为

    theme: 重命名后的名字

    #注意，冒号后必须添加一个空格

Hexo 部署至Github

  • 进入GitHub

  • 新建一个项目

  • 项目名称填为:你的用户名.github.io

    配置 deploy

    获取Github token

    • 进入Github
    • 点击右上角的头像
    • 点击_Settings_
    • 点击_Developer settings_下的_Personal access tokens_
    • 在_Token description_下方填写名字
    • 将下方的单选框全部勾上
    • 最后点击_Generate token_
    • 将那串token复制

    修改Hexo部署设置

    • 打开博客根目录的_config.yml

    • 搜索deploy

    • 将deploy: 以及下方的repo改为

      deploy:
      type: git
      repo: https://你的token@github.com/你的用户名/你的用户名.github.io
      branch: master

    • 进行Hexo“三连”

Hexo三连

hexo clean

hexo g

hexo d

hexo clean 指清除缓存和生成的静态页面

hexo g 生成静态页面

hexo d 部署

The End
